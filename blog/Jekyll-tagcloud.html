Jekyll-tagcloud
12.07.2012

jekyll-tagcloud is a little hack to create a tag cloud and their respective posts for your Jekyll generated site hosted in GitHub Pages, using Liquid and JavaScript (jQuery); without plugins. jekyll-tagcloud uses a logarithmic assessment (very simple) in order to weight tags.

You can do the same using Jekyll plugins, but that means you should have to run Jekyll locally and post the produced files into your repo (because GitHub Pages does not allow Jekyll plugins). I don’t like that solution, hence jekyll-tagcloud.
