Turing Machine Simulator

About

A simple Turing Machine simulator written in Java for my Automata and Formal Languages class.

Usage

To run tms

$ java tm.Simulator input.txt

where input.txt has the format:

aaaa
q1, a - q2, b, R
q1, b - q1, b, R
q1, # - qf, #, L
