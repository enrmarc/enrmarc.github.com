Turing Machine simulator

About

A simple Turing Machine simulator written in Java with a command-line interface.

Usage

To run tms

$ java tm.Simulator input.txt

where input.txt has the format

 
aaabbb
q1, a -> q2, #, R
q2, a -> q2, a, R
q2, b -> q2, b, R
q2, # -> q3, #, L
q3, b -> q4, #, L
q4, a -> q4, a, L
q4, b -> q4, b, L
q4, # -> q1, #, R
q1, # -> qf, #, R

The first line is the input string and the next lines are the transitions. The states are labeled starting from q1. The final state is qf, and the blank symbol is #.

To run the tms step by step use the s option:

$ java tm.Simulator input.txt s
